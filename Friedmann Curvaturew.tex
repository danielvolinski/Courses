{\mu\nu}=\mathrm{d}\mathrm{\eta}_{\mu\nu}=0$, just:
$$\omega_{\mu\nu}=-\omega_{\nu\mu}$$ so there are only
six 1-forms $\omega_{\mu\nu}$ to be found.

is to \textit{guess} a solution (because this is so often
quicker than using systematic methods) to the equations
$0=\mathrm{d}\omega^\mu+\omega_{\hphantom{\mu}\nu}^\mu
\wedge\omega^\nu$ in which the $\omega^\nu$ and thus also
$\mathrm{d}\omega^\mu$ are known, and $\omega_{\hphantom
{\mu}\nu}^\mu$, are unknown. The solution $\omega_{\hphantom
{\mu}\nu}^\mu$ is known to be unique; so guessing (if it
leads to any answer) can only give the right answer.
